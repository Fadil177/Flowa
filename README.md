# Flowa

Flowa is an MVP for a warm, low-pressure companion app for people who stutter. The focus is not fluent speech, but courage, reflection, self-acceptance, and visible everyday progress.

## Structure

- `frontend`: dependency-free single page app inspired by the Base44 prototype style.
- `backend`: dependency-free Node.js API with local JSON persistence.
- `docs`: product and technical source documents.

## Run Locally

```bash
npm run dev
```

The frontend runs at `http://localhost:5173`.
The backend API runs at `http://localhost:4000/api`.

## Test

```bash
npm test
```

## MVP Scope

- Login and onboarding flow
- Challenge library
- Challenge preparation and post-reflection
- Tension scale before and after challenges
- Success journal
- Breathing exercises
- Affirmations
- Education articles and quiz prompts
- Progress dashboard
- Export and delete endpoints for personal data

## Product Note

Flowa is not medical treatment and does not replace speech therapy, psychotherapy, or medical care. The app avoids promises of curing stuttering and frames progress as courage, reflection, and self-kindness.
